## About The Project

`@rster/worker-http-server` is a package of the `rster` project. It provides a rster-worker for http-server, its a convinient way to deploy your rster backend to the web.
Find tutorials and documentation on the [rster website](https://nsc-de.github.io/rster/). The package is available on [npm](https://www.npmjs.com/package/@rster/worker-http-server).

[rster repository](https://github.com/nsc-de/rster)
·
[rster package on npm](https://www.npmjs.com/package/@rster/worker-http-server)
·
[rster documentation](https://nsc-de.github.io/rster/)
